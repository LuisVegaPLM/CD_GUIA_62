Refrigerador para cadáveres dos gavetas - (AIRHO)

DESCRIPCIÓN:

Equipo para preservar cadáveres a temperatura de +2 a +4°C, puede fabricarse con varias capacidades de cuerpos, cerrojos especiales de tambor aisladas, bandejas para cuerpos, sistema de correderas telescópicas embaleradas para soportar marcos de acero inoxidable con jaladeras integradas. Corriente eléctrica 120 V/60 HZ. 0 220 V/60 HZ.

Fabricada totalmente de acero inoxidable calibre 20, sistema de refrigeración libre de CFC con compresor de acuerdo a equipo, condensador y difusor de aire frío, con aislamiento a base de páneles de espuma de poliuretano de alta densidad, control de alarmas visible y audible en caso de falta de suministro eléctrico, controlador digital de temperatura programable.
