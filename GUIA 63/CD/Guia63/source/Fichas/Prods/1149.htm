Tanque terapéutico - HWX 300

DESCRIPCIÓN:

• Superficie acojinada de bajo impacto con una excelente tracción.

• Velocidad variable con incrementos suaves de 0.1 a 10 mph en incrementos de 1 mph.

• Ideal para entrenamiento de marcha y actividades deportivas específicas.

• Jet de resistencia:

- Velocidad variable y direccionable.

- Manguera para masaje profundo.

• Profundidad variable:

- Profundidad variable de 1” hasta 50” con el toque de un botón para retar a los atletas o cambiar a los pacientes de un ejercicio en agua a uno de tierra.

• Controles con pantalla touch:

- Panel con pantalla touch para controlar las funciones de la caminadora (panel resistente al agua).

• Plataforma estándar:

- 8” alto x 35” ancho x 80” largo que promueve un acceso seguro a la cámara.

• Peso:

- Tanque de recuperación: 750 galones (2,837 litros).

- Software para controlar las funciones.
